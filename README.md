# Aureon AI Customer Support & Lead Qualification Assistant

## Overview
An AI-powered customer support and lead qualification assistant built for businesses using n8n and Google Gemini.

## Features
- AI customer support
- Lead qualification
- Google Sheets CRM
- Gmail automated follow-up
- Structured output parsing
- Pinecone knowledge base
- Conversation memory

## Tech Stack
- n8n
- Google Gemini
- Pinecone
- Gmail API
- Google Sheets API

## Workflow
1. Customer sends an inquiry
2. AI analyzes the request
3. AI qualifies the lead
4. Lead is saved to Google Sheets
5. Customer automatically receives a personalized email

## Business Value
- Faster response times
- Automatic lead capture
- Better lead qualification
- Reduced manual work
